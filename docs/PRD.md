# PRD (Product Requirements Document) - NexusPoint

## 비전

> "모든 사람이 30초 만에 자신만의 프리미엄 명함과 홈페이지를 가진다"

명함은 단순한 종이 조각이 아니라 첫인상을 결정짓는 브랜딩 도구입니다.
NexusPoint는 디지털 프로필 + 물리적 명함 + AI 콘텐츠 생성을 하나로 묶어,
누구나 쉽게 자신을 프로답게 소개할 수 있는 환경을 만듭니다.

---

## 핵심 기능

### 1. AI 커리어 스토리텔링 (Before/After 변환)

**Before**: "5년 경력 개발자, React와 Node.js 사용"
**After**: "풀스택 엔지니어 | 사용자 경험을 코드로 설계하는 5년 경력의 소프트웨어 아키텍트. React 생태계와 Node.js 기반 서비스로 3개 스타트업의 제품 성장을 이끌었습니다."

- Claude AI 기반 3단계 온보딩 인터뷰
- 진실 기반 포장 (사실을 왜곡하지 않고 가치를 극대화)
- 5가지 톤 선택: 전문적 / 창의적 / 따뜻한 / 대담한 / 글로벌
- 섹션별 재생성, 버전 히스토리, 인라인 편집

### 2. 개인 프로필 홈페이지 (QR 스캔 → 랜딩)

- `nexuspoint.me/u/{slug}` 커스텀 URL
- Pro 이상: 서브도메인 (`{slug}.nexuspoint.me`)
- Premium: 커스텀 도메인 연결
- ISR(Incremental Static Regeneration)로 빠른 로딩
- 모바일 최적화 반응형 디자인
- vCard 다운로드, SNS 공유 버튼
- 실시간 조회수 및 방문 통계

### 3. 명함 디자인 에디터 (템플릿 기반)

- Fabric.js 기반 템플릿 편집기
- 20+ 프로페셔널 템플릿 (무료 3종, Pro 전체)
- QR 코드 자동 삽입 (위치/크기/색상 조절)
- 양면 디자인 지원
- 실시간 미리보기 (앞/뒤)
- 인쇄용 고해상도 PDF 출력 (300 DPI)
- 색상 팔레트, 폰트 커스터마이징

### 4. 명함 인쇄 주문 (실물 배송)

- 디자인 완성 후 원클릭 주문
- 재질 선택: 아트지 / 코팅지 / 리넨 / 메탈
- 수량: 50장 / 100장 / 200장 / 500장
- 후가공: 무광 코팅, 유광 코팅, 양각, 홀로그램 (선택 추가)
- 배송: 5~7일 영업일 (빠른 배송 2~3일 옵션)
- 주문 상태 실시간 추적

## 사용자 여정

```
1. 가입 (Google/Kakao/Naver OAuth)
        ↓
2. AI 온보딩 인터뷰 (3단계, 약 3분)
   - 기본 정보 (이름, 직함, 회사)
   - 커리어 스토리 (경력, 성과, 강점)
   - 브랜딩 취향 (톤, 스타일, 타겟 인상)
        ↓
3. AI 스토리텔링 결과 확인 및 편집
   - headline, tagline, bio 자동 생성
   - 경력별 향상된 설명
   - 섹션별 재생성 가능
        ↓
4. 프로필 홈페이지 커스터마이징
   - 테마 선택, 섹션 순서 조절
   - SNS/포트폴리오 링크 추가
   - 프로필 사진 업로드
        ↓
5. 명함 디자인
   - 템플릿 선택 → 세부 편집
   - QR 코드 자동 연결 (프로필 URL)
   - 앞뒤 확인
        ↓
6. QR 연결 확인 + 공유
   - QR 스캔 → 개인 페이지 이동 확인
   - 링크 공유, vCard 다운로드
        ↓
7. 인쇄 주문 (선택)
   - 재질, 수량, 후가공 선택
   - 결제 → 배송지 입력 → 주문 완료
```

## 요금제

### Free (무료)
- 기본 프로필 1개
- AI 스토리텔링 월 3회
- 기본 템플릿 3종
- 워터마크 표시 ("Powered by NexusPoint")
- QR 코드 생성 (커스터마이징 불가)
- 기본 통계 (총 조회수)

### Pro (9,900원/월 | 99,000원/년)
- AI 스토리텔링 무제한
- 전체 템플릿 (20종+) 이용
- QR 코드 커스터마이징 (색상, 로고 삽입)
- 상세 방문 통계 (소스, 디바이스, 지역)
- 서브도메인 (`{slug}.nexuspoint.me`)
- 워터마크 제거
- 이메일 지원

### Premium (19,900원/월 | 199,000원/년)
- Pro 모든 기능 포함
- 커스텀 도메인 연결 (최대 3개)
- 프리미엄 템플릿 (메탈, 홀로그램 등)
- 프리미엄 명함 인쇄 10% 할인
- 우선 지원 (24시간 이내 응답)
- 프로필 비밀번호 보호 옵션

### Business (49,900원/월 | 499,000원/년, 최대 5인)
- Premium 모든 기능 포함 (5인)
- 팀 관리 대시보드
- 브랜드 가이드라인 적용 (공통 색상/폰트)
- 화이트라벨 (NexusPoint 브랜딩 제거)
- 명함 일괄 주문 및 관리
- 전담 계정 매니저
- API 액세스

## 성장 전략

### 바이럴 메커니즘
```
명함 받은 사람이 QR 스캔
        ↓
개인 프로필 페이지 방문
        ↓
하단에 "Powered by NexusPoint" 배지
        ↓
배지 클릭 → 가입 페이지 (레퍼럴 코드 포함)
        ↓
신규 가입 → 기존 사용자에게 크레딧 지급
```

### 레퍼럴 프로그램
- 레퍼럴 1명 가입 → 1개월 Pro 연장
- 레퍼럴 3명 가입 → 1개월 Premium 업그레이드
- 레퍼럴 링크 공유 기능 (대시보드 내)

### 콘텐츠 마케팅
- "AI가 내 직함을 바꿔봤더니..." 시리즈 (SNS 바이럴)
- 업종별 Before/After 템플릿 공유
- 명함 디자인 트렌드 블로그

### B2B 확장
- 스타트업 팀 일괄 도입 (Business 플랜)
- 대학교 취업지원센터 파트너십 (취준생 패키지)
- 행사/컨퍼런스 스폰서 (참가자 명함 제작)
- 부동산/보험/금융 등 명함이 필수인 업종 공략

## 제약 및 가정

- **인쇄 파트너**: 초기엔 3rd party 인쇄 업체 API 연동 (직접 생산 X)
- **배송**: 국내(한국) 한정 (해외 배송 추후 검토)
- **AI 비용**: 무료 플랜은 월 3회 제한으로 AI 비용 통제
- **도메인 연결**: Premium 이상에서만 제공 (SSL, DNS 설정 지원)
- **명함 심의**: 불법적/혐오적 콘텐츠 자동 필터링 (Claude Moderation 활용)
