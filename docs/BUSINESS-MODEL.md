# BUSINESS-MODEL - NexusPoint 비즈니스 모델

## 비즈니스 개요

**포지셔닝**: "디지털 프로필 + 물리적 명함 + AI 콘텐츠 생성" 3개 시장의 교차점
**수익 모델**: 하이브리드 (SaaS 구독 + 명함 인쇄 판매)
**타겟 시장**: 국내 (1차), 글로벌 한인 디아스포라 (2차)

## Unit Economics

### 고객 획득 비용 (CAC)
- **오가닉 (바이럴/SEO)**: ~2,000원/유료 전환
- **소셜 미디어 광고**: ~8,000원/유료 전환
- **유료 채널 평균 CAC**: ~5,000~15,000원

### LTV 계산 (Pro 플랜 기준)
```
평균 구독 유지 기간: 24개월 (가정)
월 구독료: 9,900원
명함 주문 평균: 분기 1회 × 평균 30,000원 = 10,000원/월
평균 ARPU: 9,900 + 10,000 = 19,900원/월

LTV = 19,900원 × 24개월 = 477,600원
순 LTV (마진 60% 가정) = 286,560원

LTV/CAC = 286,560 / 10,000 = 28.6x ✅ (목표: 3x 이상)
```

## 바이럴 성장 루프

```
┌─────────────────────────────────────────────────┐
│                 바이럴 루프                        │
│                                                  │
│  명함 제작 → QR 인쇄 → 명함 배포                  │
│       ↑                    ↓                     │
│   신규 가입            QR 스캔                    │
│       ↑                    ↓                     │
│   레퍼럴 클릭       프로필 페이지 방문             │
│       ↑                    ↓                     │
│   가입 페이지   "Powered by NexusPoint" 배지       │
│       ↑                    ↓                     │
│       └────── 배지 클릭 ────┘                     │
└─────────────────────────────────────────────────┘

예상 바이럴 계수 (K-factor):
- 명함 100장 배포 → 평균 20% QR 스캔 = 20회 방문
- 방문자의 10% 가입 = 2명 신규 가입
- 2명 중 5% 유료 전환 = 0.1명 신규 유료
→ K-factor ≈ 0.1 (낮지만 오가닉 유입 + CAC 절감 효과)
```

## 레퍼럴 프로그램

### 구조
- **레퍼럴 1명 가입** → 기존 사용자에게 1개월 Pro 연장 크레딧

## B2B 확장 전략

### 타겟 세그먼트

1. **스타트업 팀 (5-20인)**
   - 통일된 브랜드 명함이 필요
   - Business 플랜 + 일괄 명함 주문
   - 영업: LinkedIn/Product Hunt 타겟팅

2. **부동산/보험/금융 설계사**
   - 명함이 영업의 핵심 도구인 업종
   - 월 명함 재주문 수요 높음
   - 영업: 해당 업종 커뮤니티/카페

3. **프리랜서/1인 크리에이터**
   - 개인 브랜딩 중요
   - Pro 플랜 + 인쇄 정기 구독
   - 영업: 클래스101, 크몽 등 플랫폼 파트너십

4. **취업 준비생 (대학생)**
   - AI 자기소개 기능 강조
   - 졸업 시즌 집중 마케팅
   - 채용 플랫폼/취업지원센터 제휴

5. **행사/컨퍼런스 참가자**
   - 이벤트 전 대량 명함 수요
   - 이벤트 주최사 파트너십
   - 패키지: 이벤트명 + 날짜 특별 디자인
